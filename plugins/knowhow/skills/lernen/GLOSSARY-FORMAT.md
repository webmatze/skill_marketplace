# GLOSSARY.md Format

`GLOSSARY.md` ist die kanonische Sprache dieses Lern-Workspaces. Alle Erklärungen, Übungen und Lern-Records sollen seiner Terminologie folgen. Der Aufbau ist selbst Teil des Lernens: ein Konzept in eine knappe Definition zu pressen, ist ein Beleg dafür, dass der User es verstanden hat.

## Struktur

```md
# {Thema} Glossar

{Ein bis zwei Sätze über das Thema, das dieses Glossar abdeckt.}

## Begriffe

**Hypertrophie**:
Muskelwachstum, ausgelöst durch mechanische Spannung und metabolischen Stress über wiederholte Trainingseinheiten hinweg.
_Vermeiden_: Bulking, dick werden

**Progressive Überlastung**:
Die Anforderung an einen Muskel systematisch über die Zeit steigern — über Last, Volumen oder Intensität.
_Vermeiden_: Härter rangehen, Level-up

**RPE (Rate of Perceived Exertion)**:
Selbsteinschätzung 1–10, wie hart sich ein Satz angefühlt hat, wobei 10 Versagen ist und 8 bedeutet, dass noch zwei Wiederholungen drin gewesen wären.
_Vermeiden_: Effort-Score, Intensitäts-Score
```

## Regeln

- **Einen Begriff erst aufnehmen, wenn der User ihn versteht.** Das Glossar ist eine Sammlung verdichteten Wissens, kein Wörterbuch, das der User zum Lernen liest. Wenn der User gerade erst ein Konzept kennengelernt hat: warte, bis er es korrekt verwenden kann, bevor du es aufnimmst.
- **Sei meinungsstark.** Wenn mehrere Wörter für dasselbe Konzept existieren, wähle das beste und liste die anderen als zu vermeidende Aliase. So komprimiert sich Sprache.
- **Definitionen knapp halten.** Ein bis zwei Sätze. Definiere, was der Begriff IST, nicht was er tut oder wie man ihn anwendet.
- **Begriffe des Glossars innerhalb der Definitionen verwenden.** Sobald ein Begriff im Glossar steht, soll er überall bevorzugt verwendet werden — auch in anderen Definitionen. Das macht komplexere Begriffe später leichter zu fassen.
- **Bei natürlichen Clustern Unterüberschriften nutzen** (z. B. `## Anatomie`, `## Programmierung`). Eine flache Liste ist okay, wenn die Begriffe zusammenhängen.
- **Mehrdeutigkeiten ausdrücklich markieren.** Wenn ein Begriff im breiteren Fach lose verwendet wird, halte die Festlegung fest: „In diesem Workspace meint ‚Satz' immer einen Arbeitssatz — Aufwärmsätze werden separat geführt."
- **Mit zunehmendem Verständnis überarbeiten.** Eine Definition, die der User in Woche eins geschrieben hat, kann in Woche sechs falsch sein. Überschreibe sie — keine veralteten Einträge stehen lassen.
